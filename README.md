# 📖 谜典 · The Enigma Codex

> 献给所有曾与 DODO 冒险队同行的你 | 《查理九世》主题数字档案馆

**谜典**是一个关于《查理九世》系列作品的**粉丝纪念网站**。全静态页面，已适配 PC / 手机，支持 GitHub Pages 一键部署。  
网站收录了 1–16 册主线档案、人物志、奇物图鉴、箴言壁和馆志，致敬那段小有遗憾却闪闪发光的童年。


## ✨ 功能页面

| 页面 | 内容 |
|------|------|
| **序厅** 🏛️ | 首页，3D 旋转书柜 + 进入档案馆入口 |
| **墨香纪年** 📚 | 16 册剧情卡片，含豆瓣评分、简介、人物表，点击弹详情 |
| **冒险者名录** 🧑‍🤝‍🧑 | 人物墙（查理、墨多多、虎鲨等），点击查看能力雷达图 + 档案 |
| **奇物陈列室** 🔮 | 道具轮播 + 搜索/筛选（前8册/后8册），展示关键物品 |
| **箴言壁** 💬 | 经典语录分类展示 + 访客留言（前端演示，无后端） |
| **馆志** 📜 | 档案馆建站初心、大事记、导航链接 |


## 🛠️ 技术栈

- 纯 HTML5 / CSS3 / JavaScript（无框架）
- CSS 3D 旋转木马（首页书展）
- Canvas 雷达图（人物能力）
- Swiper.js 卡片轮播（奇物陈列室）
- 响应式设计 + 粒子背景 + 毛玻璃效果


## 📂 项目结构
